# Screenshots

The README's Gallery section references the four images below. Drop captures
here with these exact filenames and they'll appear automatically.

| File | Scene (cycle with `[` / `]`) | Suggested capture |
|---|---|---|
| `atrium.png`       | Scene 10 — Atrium       | Mid-room view showing the iridescent orb, marble columns, water pool, and a torch in frame |
| `pbr_lab.png`      | Scene 9 — PBR Lab       | Slightly above the metallic/roughness sphere grid so the IBL skybox is visible |
| `light_garden.png` | Scene 2 — Light Garden  | Low angle showing several colored point lights orbiting the checker ground |
| `water_pond.png`   | Scene 6 — Water Pond    | Side-on shot catching the Fresnel highlight on the animated water |

**Size**: 1280 × 720 PNG is a good default — large enough to read on a
README, small enough that the repo doesn't bloat. JPEG (`.jpg`) is fine too
if you update the references in `../../README.md`.

**How to capture**: launch `tyro`, press `]` until you reach the target
scene, frame the shot with WASD + mouse, then use your OS's screenshot tool.
Press `H` to hide the stats overlay first if you want a clean image.
